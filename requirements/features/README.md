# Future Features

Detta är en platshållare för framtida feature-specifikationer som går utanför MVP-scopet.

Feature-dokument som läggs här bör följa samma struktur som MVP-dokumenten:
- Tydligt syfte
- Kontext och motivation
- Teknisk beskrivning
- UX-flöden
- Länkar till relaterade dokument
